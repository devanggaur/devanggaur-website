## Local Development

To run locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/devanggaur/devanggaur-website.git
   ```

2. Open `index.html` in a web browser, or use a local server:
   ```bash
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```
